# winning-cs-web-platform

卫宁健康长沙研发中心前端团队文档及公共组件平台

1. 安装模块
```
npm install or yarn install
```

2. 开发启动
```
npm run docs:dev
```

3. 构建打包
``` 
npm run docs:build
```
4. 浏览器预览
open http://localhost:2333
